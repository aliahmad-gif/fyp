# SmartTailor Dashboard

This project is now connected to Firebase.

## How to Run Locally

Since this project uses **JavaScript Modules** (for Firebase integration), you cannot run it by simply double-clicking the HTML files. You need a local web server.

### Option 1: VS Code Live Server (Recommended)
1. Open this folder in VS Code.
2. Install the **"Live Server"** extension.
3. Click the **"Go Live"** button at the bottom right of the editor.

### Option 2: Python (if installed)
Open your terminal in this folder and run:
- for Python 3: `python -m http.server 8000`
- for Python 2: `python -m SimpleHTTPServer 8000`
Then go to `http://localhost:8000` in your browser.

### Option 3: Node.js (if installed)
Run `npx serve` in this folder.
